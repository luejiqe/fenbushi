🚀 快速开始
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1️⃣ 安装依赖
   pip install -r requirements.txt

2️⃣ 环境测试
   python quickstart.py

3️⃣ 准备数据
   将Animals90数据集放入 data/train 和 data/val 目录
   每个类别一个子文件夹

4️⃣ 开始训练
   python src/train.py --train_dir data/train --val_dir data/val

5️⃣ 评估模型
   python src/evaluate.py --checkpoint checkpoints/checkpoint_best.pth \
                          --test_dir data/test

6️⃣ 预测图片
   python src/predict.py --checkpoint checkpoints/checkpoint_best.pth \
                         --image your_image.jpg
